
## 3. 성공 기준 (9시간 MVP)

### 3.1 기능 완성도

```
✅ 코어 플로우 동작:
├─ 사용자 등록 → 로그인 → 공고 조회 → 지원 → 상태 확인
├─ API 응답 <200ms
├─ 에러 처리 (잘못된 입력, 인증 실패)
└─ 기본 UI (모바일 친화적)
```

### 3.2 기술 스택

```
Backend: FastAPI + PostgreSQL + Redis
Frontend: React + TypeScript + Axios
Infra: Docker + docker-compose
Auth: JWT (간단 버전)
```

### 3.3 확장성

```
🔄 이후 추가 가능:
├─ MLM 보상 로직
├─ 세무 API 연동
├─ 실시간 알림
└─ 관리자 대시보드
```
